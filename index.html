ImageXD: Image Processing Across Domains

An interdisciplinary (un)workshop

Schedule

Day 1: Instruction, with a focus on the scikit image library

Day 2 : Talks about image processing pipelines and a mapping of tools/techniques/algorithms in different domains

Day 3 : Hacking. We hope that we can identify common issues that face multiple domains in Day 2 so that we can use all of our collective efforts to work on solutions in Day 3.
